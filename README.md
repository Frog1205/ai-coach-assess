## 产品理念

AI 时代的杠杆**在体系而非工具**,在**陪跑而非课程**。

通用 AI 课程让人"学了很多但用不起来",ATLAS 反着做:
- 先用 5 分钟 **20 题诊断** 把你定位到 5 个段位之一
- 再按段位匹配 **1v1 私享陪跑方案**,深度介入你的真实业务
- 最终目标:让 AI 成为你的**战略杠杆**,而非工具栈里又一个 tab

- DeepSeek V4(精灵问答)
- 题库版本化:`qbank/*.json` + 管理端可切换激活版本
